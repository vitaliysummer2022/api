1. Установить программу XAMPP (если не установлена)
2. Запустить Apache и MySQL в панеле XAMPP
3. Загрузить API с GitHub (https://github.com/vitaliysummer2022/api.git)
4. Перенести тестируемый API в папку htdocs (находится в папке XAMPP)
5. Открыть документацию (файл doc.html)
6. Открыть Postman
7. Приступить к отправке API-запросов
